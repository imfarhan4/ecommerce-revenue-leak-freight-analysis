# E-Commerce Revenue Leak & Freight Cost Analysis

## Project Overview

This project analyzes e-commerce business performance using SQL, Python, and Power BI. The analysis focuses on revenue performance, revenue leakage, and freight cost patterns.

## Project Workflow

### SQL

* Created a consolidated view from multiple e-commerce tables
* Calculated Total Revenue
* Identified the highest revenue-generating product category
* Identified the top revenue-generating state
* Analyzed monthly revenue trends

### Python

* Performed data cleaning and preprocessing
* Conducted exploratory data analysis (EDA)
* Analyzed revenue trends, cancellations, customer reviews, and freight costs
* Generated business insights and visualizations

### Power BI

Built an interactive dashboard consisting of:

#### Revenue Overview

Analyzes revenue trends, orders, customer ratings, product categories, and regional sales.

#### Revenue Leak Analysis

Identifies revenue losses caused by cancellations, delivery delays, and customer dissatisfaction.

#### Freight Analysis

Evaluates shipping costs across products and regions to understand freight cost patterns.

## Tools Used

* SQL
* Python (Pandas, Matplotlib, Seaborn)
* Power BI
